بازی حدس عدد

یک بازی ساده تحت وب که در آن بازیکن باید عددی بین ۱ تا ۲۰ را حدس بزند.

ویژگی‌ها:

شمارش امتیاز بازیکن

ثبت بالاترین امتیاز

ارائه راهنمایی برای حدس عدد پنهان

قابلیت ریست بازی بدون حذف بالاترین امتیاز


تکنولوژی‌ها:

HTML

CSS

JavaScript


نحوه اجرا:

1. این مخزن را به دستگاه خود کلون کنید.


2. فایل index.html را در مرورگر خود باز کنید تا بازی را شروع کنید.



Guess Number Game

A simple web-based game where the player guesses a number between 1 and 20.

Features:

Track the player's score

Record the highest score

Provide hints for guessing the hidden number

Reset the game without deleting the highest score


Technologies:

HTML

CSS

JavaScript


How to Run:

1. Clone this repository to your local machine.


2. Open the index.html file in your browser to play the game.
